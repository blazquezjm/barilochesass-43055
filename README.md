# Turismo Bariloche

> Este proyecto es sobre la ciudad de San Carlos de Bariloche, Río Negro, Argentina. Muestra sus atractivos naturales, gastronomía y puntos turísticos de la ciudad.

## Herramientas utilizadas
- HTML
- CSS
	- FlexBox
	- Grids
- SASS
- GIT
- GITHUB
